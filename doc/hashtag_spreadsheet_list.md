## 使用方法

1. **投稿タイプを決定**（デモアプリ紹介、Tips、事例など）
2. **対応するハッシュタグセットをコピー**
3. **業界特化が必要な場合は追加**
4. **合計11-15個になるよう調整**

## ハッシュタグ選択ルール

- **基本セット（5-7個）** + **投稿タイプ別（3-5個）** + **業界特化（2-3個）**
- **日本語と英語のバランス**: 60%日本語、40%英語
- **大中小ハッシュタグのミックス**: 大規模20% + 中規模60% + 小規模20%
- **地域ハッシュタグは必須**: #japan #nagoya は毎回含める
